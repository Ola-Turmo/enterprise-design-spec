# Stitch Design Migration

Use this skill to expand a Stitch-only `DESIGN.md` into the full enterprise standard.

## Migration path

1. Keep the original narrative guidance.
2. Extract reusable values into `tokens/`.
3. Create manifests for the highest-value asset families first.
4. Create `docs/assets/` pages for approved variants.
5. Add CI validation and a generated catalog.

## Priority order

- logos
- color and type foundations
- social banners
- email graphics
- print masters
- motion assets
