# Nexus Brand Guidelines

## Positioning

Nexus is a modern technology brand that bridges complexity and simplicity. Our visual identity should feel approachable yet authoritative, innovative yet reliable.

## Logo system

### Primary logo

The horizontal lockup is the default for all contexts with sufficient width.

### Secondary logo

The stacked (vertical) lockup is used when horizontal space is constrained.

### Symbol only

The N-symbol may be used alone only when:
- The full logo has been established in the same view
- Space is extremely constrained (favicons, avatars)
- It appears at minimum 32×32px

### Clear space

Minimum clear space around the logo equals the height of the N-symbol on all sides.

### Minimum sizes

- Digital: 120px width (horizontal), 80px height (stacked)
- Print: 25mm width (horizontal), 17mm height (stacked)

## Color use

### Primary palette

- **Violet** is our brand signature. Use it for primary actions, links, and brand moments.
- **Slate** is our neutral foundation. Use it for text, backgrounds, and UI chrome.

### Semantic colors

- **Emerald**: Success states, positive actions, growth metrics
- **Rose**: Error states, destructive actions, urgent alerts
- **Amber** (if defined): Warning states, caution, pending actions

### Constraints

- Never use violet text on violet backgrounds
- Never use slate-500 or lighter for body text
- Always maintain minimum 4.5:1 contrast ratio for text

## Typography

### Hierarchy

- **Display** (size 1000+): Hero sections, major headings
- **Heading** (size 700-900): Page titles, section headers
- **Title** (size 500-600): Card titles, subheaders
- **Body** (size 400): Primary content, descriptions
- **Small** (size 200-300): Captions, metadata, helper text

### Rules

- Maximum two font weights per page (normal + semibold, or medium + bold)
- Line length: 60-80 characters for body text
- Never use letter-spacing wider than `wide` for body text

## Imagery

### Photography

- Natural, unposed moments
- Diverse representation
- Warm lighting, cool shadows
- Avoid stock-photo aesthetics

### Illustration

- Geometric, abstract forms
- Violet as primary accent
- Limited palette (3 colors max)
- Consistent stroke weights

## Social and campaign rules

- Profile images: Symbol only, centered on violet background
- Cover images: Full brand lockup with clear space
- Campaign graphics: Always include symbol + wordmark

## Accessibility

- All digital assets: WCAG 2.2 AA minimum
- Motion: Always provide reduced-motion alternative
- Color: Never use color as sole information carrier
- Text: Minimum 16px for body, 14px for UI labels

## Governance

- **Brand owner**: @nexus/design-team
- **Review cadence**: Quarterly brand audit
- **Escalation**: VP of Design for major brand changes
- **Requests**: Open issue in design-system repo
